Entule ZIP Release

Included files:
- Entule.app
- Open Entule.command

Recommended way to launch:
1. Double-click Open Entule.command.
2. The helper clears the downloaded quarantine flag and opens Entule.
3. If you prefer, move Entule.app into Applications first.

If macOS still blocks the app:
- Right-click Entule.app and choose Open.
- Or run: xattr -cr /path/to/Entule.app
