Entule ZIP Release

1. Drag `Entule.app` into your `Applications` folder.
2. Open `Applications`.
3. Right-click `Entule.app`.
4. Click `Open`.
5. If macOS shows a warning, click `Open` again.
6. If macOS still blocks the app, open `System Settings`.
7. Go to `Privacy & Security`.
8. Scroll down to the security section.
9. Click `Open Anyway`.

Why this happens:
- Entule is not signed with an Apple Developer ID yet.
- macOS is cautious about apps downloaded from the internet.
- After you approve it once, future launches should be normal.
