Entule ZIP Release

What to do first:
1. Open the ZIP file you downloaded.
2. You will see a folder named `Entule`.
3. Open that folder.

How to install Entule:
1. Drag `Entule.app` into your `Applications` folder.
2. If macOS asks if you want to keep the app, click `Keep`.
3. Open your `Applications` folder.
4. Right-click `Entule.app`.
5. Click `Open`.
6. If macOS shows a warning, click `Open` again.

If you still see “Apple could not verify”:
1. Open `Entule.app` once using right-click > Open.
2. If macOS still blocks it, open `System Settings`.
3. Go to `Privacy & Security`.
4. Look for the message about Entule being blocked.
5. Click `Open Anyway`.

If you downloaded the app to `Downloads` and want to launch from there:
1. Keep the app in `Downloads` until you move it to `Applications`.
2. The app should still open with the right-click method above.

Why this happens:
- Entule is not signed with an Apple Developer ID yet.
- macOS is being cautious about apps downloaded from the internet.
- The app is safe to open if you downloaded it from the official Entule release page.
