Entule for macOS

Save your work.
Reopen it later.

Entule saves a checkpoint of your apps, files, folders, and URLs, then brings them back with one launch.

Download Entule DMG View source

1. Download

Click the DMG button above. It always points to the latest release.

2. Install

Open the DMG and drag Entule to Applications.

3. Use

Launch Entule, save a session, then resume it later from the menu bar.

Need the terminal version? curl -L -o ~/Downloads/Entule.dmg https://github.com/derinbarutcu17/Entule/releases/latest/download/Entule.dmg && open ~/Downloads/Entule.dmg
