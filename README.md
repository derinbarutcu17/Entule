## Installation

Because Entule is an independent, open-source tool, macOS may show a security warning the first time you open it. That is normal for apps downloaded from the internet and not signed with an Apple Developer ID yet.

**If you already downloaded and opened the ZIP:**
1. Drag `Entule.app` into your `Applications` folder.
2. Open `Applications`.
3. Right-click `Entule.app`.
4. Click `Open`.
5. If macOS shows the warning again, click `Open` again.

**If macOS still blocks the app:**
1. Open `System Settings`.
2. Click `Privacy & Security`.
3. Scroll down to the security section.
4. Find the message about Entule being blocked.
5. Click `Open Anyway`.
6. If prompted, click `Open Anyway` one more time.

<details>
  <summary>Why does macOS do this?</summary>
  macOS is cautious about apps that are downloaded from the internet. Until Entule is signed with an Apple Developer ID, the first launch may require you to approve it manually in <code>Privacy & Security</code>.
  <br>
  After you approve it once, future launches should be normal.
</details>

## What Entule Does And How To Use It

Entule is a macOS app that saves your current work context and helps you reopen it later.

What it does:
- Saves apps, files, folders, and links into one session checkpoint
- Lets you inspect the saved session before reopening
- Supports quick save from the home screen
- Supports reusable presets for common launch setups
- Stores data locally on your Mac

How to use:
1. Install Entule by dragging `Entule.app` into `Applications`, then using `System Settings > Privacy & Security > Open Anyway` if macOS blocks the first launch.
2. Open Entule and go to `Save` to capture your current session.
3. Use `Inspect` to review saved items.
4. Click `Resume` to reopen saved items.
5. Use `Presets` to save reusable launch setups.
6. Use `Settings` only when you need permissions or reset actions.
